### **7. （8分）模拟一个“密码尝试”程序：**

    ● 正确密码是 "123456"

    ● 用户最多有 3 次输入机会

    ● 如果输入正确，输出 “登录成功” 并结束

    ● 如果 3 次都错，输出 “账户锁定”

    ● 使用 while + break 实现

```Python
# 请在此处编写代码
code = "123456"
n = 3
while n >= 1:
    code1 = input("请输入密码:")
    if code1 == code:
        print("登陆成功")
        break
    else:
        n -= 1
        print(f"密码错误,请重新尝试,还剩{n}次机会")
        if n == 0:
            print("账户锁定")
```
