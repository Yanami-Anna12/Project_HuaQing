### 参考答案与解析（先做题，再核对！）

#### 选择题解析

1. **答案：B**。解析：切片 `s[1:4]` 是**左闭右开**区间，取索引 1、2、3 的字符，即 `'y'`, `'t'`, `'h'`，组合为 `"yth"`。
2. **答案：B**。解析：`append()` 方法是将括号内的内容**作为一个整体元素**追加到列表末尾。所以 `lst` 变成了 `[1, 2, 3, [4, 5]]`，内部嵌套了一个列表，但外层列表只有 4 个元素。
3. **答案：D**。解析：字典的 `.get(key, default)` 方法在键不存在时，不会报错，而是返回指定的默认值 `"Unknown"`。
4. **答案：C**。解析：元组是不可变数据类型，创建后不允许修改其中的元素，强行赋值会抛出 `TypeError` 异常。

#### 编程题参考代码

**第1题（字符串）**

```python
def remove_vowels(s):
    vowels = 'aeiouAEIOU'
    result = ''
    for char in s:
        if char not in vowels:
            result += char
    return result

# 或者使用列表推导式（更Pythonic）
def remove_vowels_v2(s):
    vowels = 'aeiouAEIOU'
    return ''.join([char for char in s if char not in vowels])

# 测试
print(remove_vowels("Hello World"))  # Hll Wrld
print(remove_vowels("Python"))       # Pythn
```

**第2题（列表）**

```python
nums = [3, 1, 4, 1, 5, 9, 2, 6, 5]

# 初始化最大值和次大值
max_num = float('-inf')
second_max = float('-inf')

for num in nums:
    if num > max_num:
        # 发现新的最大值，旧的最大值变为次大值
        second_max = max_num
        max_num = num
    elif num > second_max and num != max_num:
        # 当前数比次大值大，但又小于最大值（严格小于）
        second_max = num

print("第二大的数是：", second_max)  # 输出 6
```

**第3题（字典）**

```python
letters = ['a', 'b', 'a', 'c', 'b', 'a', 'd']
count_dict = {}

for letter in letters:
    # 如果键存在，取出值加1；不存在则取默认值0再加1
    count_dict[letter] = count_dict.get(letter, 0) + 1

print(count_dict)  # {'a': 3, 'b': 2, 'c': 1, 'd': 1}
```

**第4题（元组）**

```python
students = [
    ("张三", 85, 90, 78),
    ("李四", 88, 76, 95),
    ("王五", 90, 88, 82)
]

max_name = ""
max_total = -1

for name, chinese, math, english in students:
    total = chinese + math + english
    print(f"{name} 的总分为：{total}")  # 顺便打印每个人
    if total > max_total:
        max_total = total
        max_name = name

print(f"最高分是 {max_name}，总分为 {max_total}")  # 最高分是 王五，总分为 260
```
