

#### 一、变量与基本运算

#### 题目1：交换两个变量的值
**描述**：  定义两个变量 `a = 10`，`b = 20`，编写代码交换它们的值，使 `a` 变为 20，`b` 变为 10。  
**要求**：不能直接写 `a = 20; b = 10`，需通过变量运算或临时变量实现。

**输出示例**（打印 a 和 b）：  
```
a = 20, b = 10
```

#### 题目2：计算圆的面积
**描述**：  从用户输入获取圆的半径（浮点数），计算并输出圆的面积（π取 3.14159）。  
**要求**：使用变量存储半径和面积，打印时保留两位小数。

**运行示例**：  
```
请输入半径：5
圆的面积为：78.54
```

### 二、条件分支语句（if-elif-else）

#### 题目3：奇偶数判断
**描述**：  输入一个整数，判断它是奇数还是偶数，并输出相应信息。

**运行示例**：  
```
请输入一个整数：7
7 是奇数
```

#### 题目4：成绩等级评定
**描述**：  输入一个百分制成绩（0~100），输出对应的等级：  

- 90 及以上 → A  
- 80 ~ 89 → B  
- 70 ~ 79 → C  
- 60 ~ 69 → D  
- 60 以下 → E  

**要求**：若输入超出 0~100 范围，提示“成绩无效”。

**运行示例**：  
```
请输入成绩：85
等级：B
```

#### 题目5：闰年判断
**描述**：  输入一个年份，判断是否为闰年。  

规则：能被4整除但不能被100整除，或者能被400整除。

**运行示例**：  
```
请输入年份：2024
2024 是闰年
```

### 三、循环语句（for / while）

#### 题目6：打印乘法口诀表
**描述**：  使用嵌套循环打印 9×9 乘法表，输出格式需对齐。

**输出示例**：  
```
1x1=1  
1x2=2  2x2=4  
1x3=3  2x3=6  3x3=9  
...（直至9x9=81）
```

#### 题目7：累加求和（while）
**描述**：  使用 `while` 循环计算 1 到 100 之间所有整数的和，并打印结果。

**输出示例**：  

```
1+2+...+100 = 5050
```

#### 题目8：猜数字游戏（综合运用）
**描述**：  程序内部随机生成一个 1~100 之间的整数（可用 `random.randint`），让用户反复猜这个数字。  

- 猜大了 → 输出“太大了”  
- 猜小了 → 输出“太小了”  
- 猜中了 → 输出“恭喜你，猜对了！”并结束循环  

**要求**：记录并输出用户猜的次数。

**运行示例**：  
```
猜数字游戏开始！（1-100之间）
请输入你的猜测：50
太小了
请输入你的猜测：75
太大了
请输入你的猜测：63
恭喜你，猜对了！你一共猜了3次。
```
